## Estructura

```
src/
├── i18n/
│   ├── es.json         Todos los textos en español
│   ├── en.json         Todos los textos en inglés
│   └── index.js        Configuración de i18next
├── components/
│   └── Navbar.jsx
├── pages/
│   ├── Home.jsx       Inicio + Sobre mí + Logros + Artículo
│   ├── Projects.jsx   Proyectos de GitHub
│   ├── Talks.jsx      Charlas y presentaciones
│   └── Contact.jsx    Contacto y redes sociales
├── App.js
├── index.js
└── index.css
```

## Cómo personalizar

 1. Textos e información
Edita **`src/i18n/es.json`** y **`src/i18n/en.json`** con:
- Nombre real
- Bio y descripción
- Logros
- Título y link del artículo
- Proyectos (título, descripción, tags, link al repo)
- Charlas (título, evento, slides, video)
- Links de contacto (email, GitHub, LinkedIn, Twitter)
